# 🏨 Hotel Room Booking System

## 📌 Description

A full-stack web application to manage hotel room bookings with date validation and conflict prevention.

## 🚀 Features

* View available hotels
* Book rooms with check-in & check-out dates
* Prevent overlapping bookings
* View all bookings

## 🛠 Tech Stack

* Frontend: React.js
* Backend: Node.js, Express.js
* Database: SQLite

## ⚙️ How to Run

1. Clone the repo
2. Install dependencies:
   npm install
3. Start backend:
   node app.js
4. Start frontend:
   npm run dev
